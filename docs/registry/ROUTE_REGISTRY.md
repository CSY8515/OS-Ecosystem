External routes use their real public Streamlit application URL directly. Redirect intermediaries, login routes, `st.switch_page`, query redirect, and meta refresh are prohibited.

Future projects receive no implementation in advance. They enter through an approved Project Registry record, 6W Metadata, Route, Status, Health, ownership, and version declaration.
