# Contract Registry

Version: v0.7.0

| Contract | Authority | Scope |
| --- | --- | --- |
| Common UI System | Architecture | concept-as-interface tokens, components, usability, accessibility |
| 6W Metadata Contract | Architecture | explanatory identity and traceability |
| Navigation Contract | Architecture | shared shell and route behavior |
| Project Registry | Registry | connected and repository-owned OS Systems |
| Capability Registry | Registry | OS Ecosystem Core capabilities |
| Route Registry | Registry | internal and external destinations |
| Status Contract | Registry | public readiness language |
| Health Contract | Capability or platform owner | safe availability evidence |
| Release Contract | Governance and Release | validation and publication evidence |

Contracts expose only approved public information. Component-specific execution and data contracts remain with their owning package. UI language reuse by Living OS, Universal Learning Engine, or Ultra Brain does not transfer ownership or Governance authority.
