# Personal Secretary Capability

Version: v1.0.0
Status: Stable
OS Ecosystem release: v0.6.0

Personal Secretary turns project-owned signals into deterministic briefings and decision support. It recommends and summarizes; it does not own source data or autonomously execute work.

## Public functions

| Function | Output |
| --- | --- |
| Daily Briefing | Today's priorities, events, reminders, and recommendations |
| Weekly Review | Progress summary and next-step recommendations |
| Monthly Review | Monthly outcomes, improvements, and recommendations |
| Smart Reminder | Ordered due and overdue reminders |
| Recommendation | Ranked next-action candidates with their basis |
| Priority | Weighted urgency, importance, goal, and deadline ranking |
| Decision Support | Criteria-weighted option comparison and recommendation |
| Notification | Deduplicated, stateful advisory notifications |
| Secretary Service | A single advisory-only synthesis entry point |

The runtime operates without AI. Living OS, Universal Learning Engine, and future projects continue to own their data.
