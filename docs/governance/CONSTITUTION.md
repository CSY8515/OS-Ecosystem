# Ecosystem Constitution

Version: v0.6.2

OS Ecosystem owns shared governance, architecture, registry, release standards, and explicitly approved common platform components without owning connected-project business data or runtime decisions.

Living OS and Universal Learning Engine retain independent code, data, versions, and release responsibility. Capabilities retain explicit package boundaries. AI Hub is owned, versioned, released, and deployed by OS Ecosystem.

The [documentation index](../README.md) is authoritative. Structural changes require validated links, regression tests, and release evidence.
