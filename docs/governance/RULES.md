# Ecosystem Rules

Version: v0.4.4

1. Projects remain independently deployable.
2. External navigation uses direct public URLs.
3. Registries expose approved identity and version metadata only.
4. Authoritative design and release documents live under docs/.
5. Each document has one authoritative location.
6. Relative internal links must pass validation.
7. Capabilities follow the [Capability Documentation Standard](../capabilities/README.md).
8. Documentation patches do not change runtime behavior.
