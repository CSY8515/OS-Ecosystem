# OS Ecosystem v0.72 Release Notes

Status: Stable
Release type: Interactive World UI Finalization

## Summary

v0.72 completes the official OS Ecosystem world-based Home UI. The supplied Official Answer remains the locked Art Direction and interface composition. No artwork, layout, Project placement, card geometry, label structure, Route, or responsive architecture changes are included.

## Included

- Official world-based OS Ecosystem main UI
- Official Answer-based final art and locked Project Seed geometry
- Living OS, Universal Learning Engine, and AI Hub full-Seed project entry UI
- Restrained Hover brightness, Project-color boundary, label, and route-arrow feedback
- Keyboard-visible Focus with information equivalent to Hover
- Immediate Active press feedback without navigation delay
- Desktop, Tablet, Mobile, reduced-motion, and accessibility finalization
- Final OS Ecosystem Overview, AI Hub, Living OS, and Universal Learning Engine navigation connections

## Preserved

- v0.7.0 Architecture and Governance responsibility boundaries
- Registry, Contract, Capability, AI Hub, and 6W Metadata behavior
- Official artwork files and Art Direction
- Direct public `*.streamlit.app` URLs for Living OS and Universal Learning Engine
- Repository-owned `?project=ai-hub` and `?view=overview` Routes
- Secret, Token, database, runtime, and personal-data non-disclosure

## Validation

- 203 automated tests across the repository and included components
- Python syntax and Streamlit startup checks
- Desktop 1440×1000, Tablet 820×1180, and Mobile 390×844 rendering
- Hover contract, actual Focus, Active contract, touch targets, navigation, and zero horizontal overflow
- Production console and resource-loading checks

## Known issues

- Streamlit Community Cloud may put independently deployed external Projects to sleep after inactivity; their direct URLs and ownership are unchanged.
- No known OS Ecosystem v0.72 release-blocking issue.
