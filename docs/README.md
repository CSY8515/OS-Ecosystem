## Management principles

1. Official documents use the narrowest `docs/` category.
2. UI, Registry, and Contract use the same classification, version, and Route language.
3. Relative internal links are automatically validated.
4. Capability documents retain the common standard and capability-owned contracts.
5. `PRINCIPLES.md` remains the approved source; the separate official six operating principles follow after source confirmation.
