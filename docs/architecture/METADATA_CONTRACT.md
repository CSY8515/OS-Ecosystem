## Rules

1. 6W is explanatory metadata, not six mandatory input controls.
2. Metadata is concise, Korean-first, and may include an official English name.
3. Every OS System and Capability registry entry has all six fields.
4. Execution, Decision, and Report records preserve actor, timestamp, target, action, method, and reason when those records exist.
5. Secret values, raw prompts, private payloads, database schemas, and internal identifiers are excluded.
6. UI summaries and registries use the same authoritative metadata source.
